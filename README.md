# Indonesia Supreme Court Crawler


How to use:
```bash
 # Install Dependencies
 $ make install
 # Build
 $ go build -o indonesia-supreme-court-crawler
 # Run Crawler and Scrapper
 $ ./indonesia-supreme-court-crawler
 # Run Crawler
 $ ./indonesia-supreme-court-crawler crawler
 # Run Scrapper
 $ ./indonesia-supreme-court-crawler scrapper
```
